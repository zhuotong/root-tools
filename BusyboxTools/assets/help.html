Busybox 无法安装的解决办法

★环境需求
您的手机必须拥有 ROOT 权限，并且保证在 /system/app 目录下，包含 Superuser.apk 这个文件（不区分大小写），以及在 /system/bin 或 /system/xbin 目录下包含 su 文件（区分大小写）
保证 su 文件的权限是 06755，若权限不正确，可能无法正常工作。
★保证权限
您可以使用以下命令，将 su 的权限设置为 06755
# cd /system/bin
# chmod 06755 su
此时您可以尝试重启手机，并重新执行本软件，大多数情况下，可以成功的安装了
★还不能安装
若是您的手机依然无法成功执行本软件，请按以下步骤输入命令，完成安装
# mount
此命令将列出手机中的分区挂载情况，找到 /system 对应的分区，例如找到以下字符串：
/dev/block/mmcblk0p25 /system ext3 ro,relatime,errors=continue,barrier=0,data=ordered 0 0
此时分区就是 /dev/block/mmcblk0p25
使用以下命令将分区重新挂载为可读写
# mount -o remount,rw /dev/block/mmcblk0p25 /system
成功挂载后，即可继续进入以下操作（若是 mount 时发现 /system 已是 rw 状态，可以跳过 remount 的步骤）。
# cd /sdcard/.busybox
# cat busybox > /system/xbin/busybox
# cd /system/xbin
# ln -s ./busybox /system/bin/busybox
# chmod 777 ./busybox
# ./busybox --install -s /system/xbin
现在您应该已经能够成功的安装 busybox 了
★上述命令也不能安装
若是按上述命令依然无法安装，请检查系统分区内是否还有足够的空间，若是空间不足，则无法将 busybox 植入系统内，这个问题有可能需要重新刷机才能解决。
